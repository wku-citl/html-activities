🎯 What Are These?

These aren't ready-made activities for specific courses—they're templates that show what's possible.

The psychology flashcards can become business term flashcards. The chemistry sorting activity can become a literary genre sorter. The timeline can display any chronological sequence.

Use AI to adapt these templates to your discipline and content in minutes.

🤖 How to Customize with AI

1
Try the activity

Click the link and interact with it to understand how it works

2
Download the HTML

Right-click on the page → "View Page Source" (or press Ctrl+U) → Select all code (Ctrl+A) → Copy (Ctrl+C)

3
Open ChatGPT or Claude

Use a prompt like this:

"I have an HTML flashcard activity about psychology concepts. I want to adapt it for my biology course to quiz students on cell organelles. Keep all the functionality (flip cards, study mode, mastery tracking) but replace the psychology content with 12 cell organelle terms and definitions. Here's the HTML: [paste code]"
4
Copy the new code

AI will generate customized HTML ready to use

5
Upload to your LMS

In Blackboard Ultra: Create → Document → Paste code into HTML content block
